Generates and displays PU Maps of every state in Nigeria

1. Make sure that you have python3 installed.
install depedencies by running:
`pip install folium flask pandas numpy flask_htmlmin``


Now to generate the data sets

1. Run `python init.py`
2. Run `python run-server.py`
Open your browser http://127.0.0.1:5000/ABIA

you can change the state name to see differnt state maps. Zoom in and Zoom out as you please.


Doing all the PUs in one file takes forever to load so opted out of generating a single map for all PUs
- You can change the generation options by looking at the source
